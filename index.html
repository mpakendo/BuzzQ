Tap into the social media buzz!

Just enter a tag and query several social media services at once. Browse through simple image galleries and a paginated twitter stream to screen the conversation!

Technical background:

This simple app takes a keyword and queries several social media services in parallel. Currently the app has integrated Twitter, Flickr, Twitpic and Instagram through their REST APIs. These are called via a simple node.js server which passes the results back to the browser app. This was made possible by the use of JQuery, JQuery-UI, Galleria, and Imtech pager. Currently the node.js app is hosted at Heroku.

This was done purely as a hobby hack. There are known bugs.
